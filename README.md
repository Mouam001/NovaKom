## 🌟 À propos de NovaKom

**NovaKom** est une startup innovante basée aux Comores, spécialisée dans l'accompagnement des entreprises et PME dans la mise en place et l'optimisation de leurs systèmes d'infrastructure informatique. Nous proposons des solutions complètes en cybersécurité, support IT, administration réseau, développement logiciel et formations spécialisées.

### Notre Mission
Sécurisez et structurez votre infrastructure IT. NovaKom accompagne les entreprises comoriennes dans la gestion et la protection complète de leur environnement numérique. Un partenaire stratégique IT & Sécurité de niveau international.

### Notre Vision
Construire la prochaine génération d'infrastructures numériques comoriennes avec une équipe d'élite de 5 professionnels passionnés, certifiés et engagés.

## 🛠️ Technologies Utilisées

### Frontend
- **React 18** - Bibliothèque JavaScript pour interfaces utilisateur
- **TypeScript** - JavaScript typé pour un code plus robuste
- **Tailwind CSS** - Framework CSS utilitaire
- **Vite** - Outil de build rapide pour le développement moderne
- **Radix UI** - Composants accessibles et non stylisés
- **Lucide React** - Icônes modernes

### Backend & Base de Données
- **Supabase** - Plateforme backend-as-a-service
- **PostgreSQL** - Base de données relationnelle

### Outils de Développement
- **ESLint** - Linting du code JavaScript/TypeScript
- **PostCSS** - Outil de transformation CSS
- **Autoprefixer** - Ajout automatique des préfixes CSS
